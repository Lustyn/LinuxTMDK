# Linux Total Miner Devkit
This repository contains scripts which aid development of Total Miner mods on Linux.

## Downloading Total Miner DLLs (TotalMinerMods repo)
```
./download_tm.sh
```

## Downloading XNA
```
./download_xna.sh
```

## Installing XNA
```
./install_xna.sh
```

## Uninstalling XNA
```
./uninstall_xna.sh
```
